## Overview

Story Weaver is a cozy, collaborative story writing platform designed for creative teams, writers, and hobbyists. Built with HTML, CSS, and JavaScript, this web app allows multiple users to contribute to a story one sentence at a time. Each user only sees the most recent part of the story and can add their own line to continue the narrative.

## Features

- **Collaborative Writing:** Add your sentence to the story and help shape its direction.
- **Editable Contributions:** Users can edit the sentences they wrote for smoother flow and corrections.
- **User-Friendly Interface:** Warm orange and black palette creates a relaxing, inviting atmosphere.
- **Session-Based Identity:** Optionally enter your name before contributing.
- **Story Timeline:** View the full history of all contributions.
- **Responsive Design:** Works on desktop and mobile.

## How It Works

1. Read the last few sentences of the current story.
2. Enter your name (optional).
3. Add your contribution by typing the next sentence.
4. Edit your sentence if you wish.
5. Start new stories anytime.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Design:** Orange & Black color palette, cozy UI
- **Data Storage:** In-memory JavaScript structures (session-based)

## Getting Started

1. Clone this repository.
2. Open `index.html` in your web browser.
3. Enjoy collaborative storytelling!

## Customization

- Change color palette in the CSS for a different vibe.
- Add a backend/API for persistent cross-user storage.
